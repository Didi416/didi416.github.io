#### Key Learning Outcomes:
Polymorphism/Inheritance/Classes/Gazebo/Multithreading/Graph Searching/Unit Tests/ROS/Nodes/Topics/Services/Rosbags

![image-left](/assets/images/PFMS_Starting.png){: .align-left} Through a series of progressively complex robotics programming assignments, I developed and implemented autonomous vehicle control systems using ROS2 and Gazebo simulation. The projects demonstrated mastery of object-oriented programming principles through the creation of control systems for multiple vehicle types. 

Starting with basic vehicle control of an Audi R8 and Husky platform, the assignments progressed to advanced implementations including a Quadcopter, featuring multithreading and path planning capabilities. The final project showcased autonomous navigation using laser-based obstacle detection and dynamic path generation. These assignments highlighted my expertise in C++ programming, robotics simulation, real-time control systems, and software architecture while emphasizing industry-standard practices in modular development and comprehensive testing.

### Skills Developed:
- **Advanced C++ Programming:** Demonstrated expertise in object-oriented programming principles including inheritance, polymorphism, abstraction, and encapsulation
- **Robotics Simulation:** Gained hands-on experience with Gazebo physics simulator and ROS2 (Robot Operating System), industry-standard tools in robotics development
- **Vehicle Control Systems:** Implemented complex control algorithms for both UGVs (Ackerman for Audi R8, and Skidsteer for Husky) and UAVs (Quadcopter)
- **Software Architecture:** Designed modular, maintainable code using interface-based programming and proper abstraction techniques
- **Real-time Systems:** Developed systems for real-time vehicle control, including motion planning and trajectory execution
- **Testing and Debugging:** Created and executed comprehensive unit tests, ensuring robust functionality across different platforms
- **Technical Documentation:** Produced clear, detailed documentation of system architecture and implementation details

This project demonstrates my ability to develop complex robotic control systems while adhering to software engineering best practices. The successful implementation showcases both technical programming skills and practical understanding of basic vehicular control systems.
